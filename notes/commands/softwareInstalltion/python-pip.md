### Pip installation:

```bash
# python 2.X should be installed
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py; python get-pip.py
```

### Pip3 installation:

```bash
# bash Advanced Package Tool
sudo apt-get install python3-pip
```
